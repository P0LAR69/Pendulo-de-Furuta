# Pendulo-de-Furuta
Este Git Hub presenta una comparación de diversas estrategias de control aplicadas a un péndulo rotacional de Furuta, simulado en el entorno de CoppeliaSim. Se evaluaron los controladores Gain Scheduling, Regulador Cuadrático Lineal (LQR) y Realimentación de Estados. El objetivo principal fue analizar y comparar el desempeño de cada controlador
